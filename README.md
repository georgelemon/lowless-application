## Loopless
**A minimal PHP framework for developing front-end web applications.** Loopless main scope is to be an API addicted and stay wired to your whatever backend apps (one or more), but it can be also used as a static & standalone app (see WikiPeep project).

Out of the box, Loopless comes with a MVC structure (without a database setup), a simple Router with dependency injection support, a Configuration Array in Laravel style and that's it. It is highly recommended for creating front-end apps, small to medium websites. Fully OOP, with Composer support - all you need is creativity. ðŸ¤ª
