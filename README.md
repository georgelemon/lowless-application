## Loopless
**A minimal PHP framework for developing fast web applications**.

Loopless provides a MVC structure containing a simple HTTP Router with dependency injection support, Symfony/Kernel, Illuminate/Filesystem and Database, a Config / Registry Array access in Laravel style and that's all!

Loopless is our first choice when it comes to create fast custom web apps targeting low traffic, private interfaces and so on &mdash; Fully Object Oriented, with Composer support.
